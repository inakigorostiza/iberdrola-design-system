Movimiento en acción

Micro-interacciones ejecutables que muestran los principios de movimiento de la marca: con propósito, tranquilas y con energía. Todas usan los tokens de movimiento y respetan prefers-reduced-motion. Consulta la guía en motion.md.

Contador · count-up

Cifra que sube hasta su valor (JS, con reduced-motion seguro).

f
0%
Energía de origen renovable
const dur = 700; // --ib-duration-slower
requestAnimationFrame(function tick(t){
  const p = Math.min((t - start) / dur, 1);
  const eased = 1 - Math.pow(1 - p, 3); // decelerate
  el.textContent = Math.round(eased * target) + suffix;
  if (p < 1) requestAnimationFrame(tick);
});
// prefers-reduced-motion → jump straight to target
